# Arch

## Upgrade the system

```shell
pacman -Syu
```

[See wiki](https://wiki.archlinux.org/index.php/pacman#Upgrading_packages)

## Install package

```shell
pacman -S <package-name>
```

### Resolving dependencies

- phonon-qt4-backend
  - **phonon-qt4-vlc**
  - phonon-qt4-gstreamer 


## Remove package

```shell
pacman -Rs <package-name>
```
