# OpenFl

## Installation

### Dependencies

- Haxe
- Neko

### Linux

```shell
# Run as root
haxelib install openfl
haxelib run openfl setup
```

#### Non-interactive installation

```
# Run as root
haxelib install openfl
echo y | haxelib run openfl setup
```
