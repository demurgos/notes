# Configuration

## User information

```shell
# As a normal user:
git config --global user.name "Your Name"
git config --global user.email "youremail@example.com"
```

Example:

```shell
# As a normal user:
git config --global user.name "Charles Samborski"
git config --global user.email "demurgos@demurgos.net"
```
