# Composer Installation

## Arch Linux

```shell
# Run as root
pacman -S composer
```
## Check the installation

```shell
# Run as a normal user
composer --version
```

`composer.phar` is called ` /usr/bin/composer` 

(PHAR: PHP Archive)

Example:

```
$ composer --version
Composer version 1.3.0 2016-12-24 00:47:03
```
